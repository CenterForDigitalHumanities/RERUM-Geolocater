Welcome to the RERUM Geocoder hosted by the Walter J. Ong, S.J., Center for Digital Humanities at Saint Louis University. This small application creates GeoJSON-LD Points as Web Annotations. You can target any Linked Open Data resource in the world and provide coordinates as supplementary data. This does not affect the original nor does it notify the resource owner of the assertion. However, a UI is provided to see the Points provided on a Leaflet map.

This application is intended to work within the scope of IIIF and encourages you to target IIIF resources in order to provide supplementary geospatial data for them. This data is meant to assert the CURRENT location of the physical object the IIIF resource type represents.

To learn more about the Walter J. Ong, S.J., Center for Digital Humanities at Saint Louis University, visit their website. To learn more about IIIF and Web Annotations, visit IIIF on the web. To learn more about GeoJSON-LD, visit this page about the standard.
